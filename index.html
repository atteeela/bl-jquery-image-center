    Description

    Centers an image by moving, cropping and filling spaces inside it's parent container. This plugin maintains aspect ratio giving it a very professional looking finish in most situations.

    Features

    • Works on IE7, 8, 9, Safari, Chrome, Firefox
    • Waits for the image to be loaded and fixes IE cached image bug
    • Supports filling entire space or keeping entire image visible maintaining aspect ratio

There is however support for inside mode which will center the image and keep it fully visible with no cropping. This is achieved by passing 'inside' as the first parameter to centerImage()
